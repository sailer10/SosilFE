# SosilFE
응용소프트웨어실습 최종과제. 던전 탈출 Mud 게임

## 목표
* 윈폼을 이용해 던전 탈출 로그라이크 게임을 구현한다.
* 시작시 랜덤한 스테이지 갯수와 크기, 몬스터를 생성한다.
* 플레이어는 상하좌우로 움직이며 포탈을 통해 스테이지를 이동, 몬스터를 피하여 탈출하면 승리하게 된다.
* 이동중 몬스터를 만나 체력이 0이 되면 패배

## 화면 구성
### 오브젝트 구분
* ▩ - 빈 공간
* ■ - 플레이어
* ∑ - 몬스터
* ● - 포탈
* ◎ - 출구
